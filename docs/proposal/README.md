## Research Focus
- Electricity price forecasting in PJM Day-Ahead Market
- 11 machine learning models comparison
- Feature fusion and model explainability using SHAP
- Multi-resolution forecasting (hourly and daily)
- Volatility analysis

## Key Research Questions
1. Which ML models achieve highest accuracy under identical conditions?
2. How does accuracy differ between hourly and daily horizons?
3. What is the impact of exogenous variables on performance?
4. How do models perform in high-volatility periods?
5. Which features contribute most (SHAP analysis)?
6. What are the computational trade-offs?

## Expected Outcomes
- <10% MAPE accuracy
- Comprehensive model comparison framework
- Explainable AI insights for electricity markets
- Reproducible research pipeline
